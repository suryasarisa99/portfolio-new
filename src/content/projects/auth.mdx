## Technical Architecture
- **Frontend:** React (Vite/CRA)
- **Backend:** Express.js (Node.js)
- **Authentication & Security:** Firebase Auth, Speakeasy (TOTP/HOTP)
- **Database:** MongoDB
- **Hosting/Deployment:** Render/Vercel/Netlify (Frontend), Render/Heroku (Backend)
- **Communication:** REST API between frontend and backend
